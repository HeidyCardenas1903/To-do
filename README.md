# To-do
Proyecto practico de universidad para materia de mantenimiento de software para la actividad de control de versiones usando Git y GitHub.

## Funcionalidades

- Crear tareas.
- Visualizar tareas.
- Editar tareas.
- Eliminar tareas.
- Marcar tareas como completadas.

## Tecnologías usadas

- HTML
- CSS
- JavaScript
- Git
- GitHub

## Integrantes

- Integrante 1: Crear tareas
- Integrante 2: Visualizar tareas
- Integrante 3: Editar tareas
- Integrante 4: Eliminar tareas
- Integrante 5: Marcar tareas completadas
